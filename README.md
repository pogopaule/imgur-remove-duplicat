# Install

```bash
git clone https://www.gitub.com/pogopaule/imgur-remove-duplicate
cd imgur-remove-duplicate
pip install -r requirements.txt
```

# Run

```bash
export IMGUR_CLIENT_ID=<insert yours>
export IMGUR_CLIENT_SECRET=<insert yours>
export IMGUR_ACCESS_TOKEN=<insert yours>
export IMGUR_REFRESH_TOKEN=<insert yours>

python remove.py

Album Id: ABCDE
OK, first I need to download all images in this album.
Done. Removing all duplicate images now...
Done. Deleted 347 images.
```
